Projects

Letter of Recommendation Tracking System for the Office of Pre-Professional Health

My capstone project that we just started working on this semester. This system allows advisors from the Office of Pre-Prefessional Health(PPHA) so that they can better track letters that students need to apply for medical school. The PPHA advisors currently use a large excel spread sheet to track all of this information. Our system that we are designing will allow easier and more efficient tracking, as well as allow students to view if their letter requests have been received.

Online dating site

This was a project that I made for my Client Side Web Development class. This website uses javascript, html, and CSS to let users go through a dating site. They can create a profile and see how it looks. They can go through men, women, or both and see everyone. They are able to like or pass on every user they see. They can also search by religion, state, and age. Users may also view their likes and remove them.

House Search with Database and Web API

This project was also for my Client Side Web development class. This project required us to create a page where users could search through houses by various fields. This also made us create a web API to interact with our database.
